# JEmalloc Stats Analyzer Usage EXAMPLES

### Sort by total requests
```bash
je-analyze ../stats3.db --mode bin_activity_analysis
```
### Sort by lock operations
```bash
je-analyze ../stats3.db --mode bin_activity_analysis_by_locks
```

Now you can use it in several ways:

#### List all tables
```bash
je-analyze stats3.db --list-tables
```
#### List only tables starting with "merged"
```bash
je-analyze stats3.db --list-tables --prefix merged
```
#### List only tables starting with "arenas"

```bash
je-analyze stats3.db --list-tables --prefix arenas
```

## more

```bash
je-analyze ../stats3.db --mode bins_analysis
je-analyze ../stats3.db --mode table --table merged_arena_stats.bins_v1 --limit 20
je-analyze ../stats3.db --table merged_arena_stats.bins_v1 --limit 20
```
